## [1.0.0] - 2026-02-09

### Added
- Initial release of Power BI Themes Library
- 6 production-ready themes:
  - corporate_dark_purple.json
  - dark_modern.json
  - foxtrot_warm.json
  - high_contrast_accessible.json
  - lavender_professional.json
  - light_professional.json
- 13 visual-specific snippets (bar, card, donut, funnel, gauge, KPI, line, maps, scatter, slicer, table, waterfall)
- Complete theme template with all properties
- Comprehensive theme properties reference documentation
- README with installation and usage instructions
- CONTRIBUTING guide for theme submissions
- MIT License

### Migration
- Migrated from PBI_Development/powerbi_themes/
- Reorganized structure: themes/, snippets/, templates/, examples/
- Source commit: [hash from PBI_Development]
